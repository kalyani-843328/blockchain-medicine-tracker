# 🏥 Blockchain Medicine Tracker

A decentralized medicine tracking system built on Blockchain to prevent counterfeit medicines and ensure supply chain transparency.

## 📌 About The Project

Counterfeit medicines are a global problem. This project uses **Blockchain technology** to:
- Track medicines from manufacturer to patient
- Prevent fake/duplicate medicines in supply chain
- Provide transparent and tamper-proof records
- Ensure patient safety

---

## ✨ Features

- ✅ Add medicine to blockchain ledger
- ✅ Track medicine at every supply chain step
- ✅ Verify medicine authenticity
- ✅ Tamper-proof records
- ✅ Decentralized — no single point of failure
